SDL game engine

This is an experimental engine framework made with pure SDL, using SDL image sprites and containing classes and methods for loading a series of scenes as defined in external .txt files and the defining of specific behaviors using a scripting template that must be compiled with the rest of the framework

View source on GitHub

OpenGL game engine

An adaptation and refinement of the SDL game engine that uses OpenGL as renderer in place of the SDL method, and can render 3D models loaded from .OBJ files and supports compiling external shaders, with a default shaders containing an implementation of a phong lighting model.

View source on GitHub
